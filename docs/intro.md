# Introduction

Welcome to the AI-Native Textbook on Physical AI & Humanoid Robotics. This comprehensive resource bridges the gap between artificial intelligence and robotics, focusing on embodied intelligence and the integration of AI systems with physical robotic platforms.

## About This Textbook

This textbook is designed for:
- Undergraduate and graduate students in Robotics, AI, and Computer Engineering
- Robotics educators and lab instructors
- Software engineers transitioning into robotics

## Module Overview

### Module 1: The Robotic Nervous System (ROS 2)
This foundational module covers ROS 2 architecture, communication patterns, Python AI agent integration, and URDF modeling for humanoid robots. You'll learn how to create the software nervous system for embodied AI systems.

## Getting Started

To get started with this textbook, make sure you have:
- Ubuntu 22.04 LTS installed
- ROS 2 Humble Hawksbill (or Iron Irwini) properly configured
- Basic Python programming knowledge

Each module includes theoretical foundations, practical lab exercises, and assessment tasks to reinforce your learning.
